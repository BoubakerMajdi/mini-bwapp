# Mini-bWAPP - A Tiny Vulnerable Web App for Learning

Inspired by bWAPP, this is a simple PHP/MySQL training app to practice web security.

## Vulnerabilities Included
- XSS (Reflected + Stored) with 3 security levels
- CSRF (Password Change)

## Security Levels
- **Low**: No protection (easy exploits)
- **Medium**: Partial filtering (realistic mistakes)
- **High**: Proper sanitization

## Setup (2 minutes)
1. Install XAMPP/WAMP
2. Copy folder to `htdocs/mini-bwapp`
3. Import `sql/comments.sql` into phpMyAdmin
4. Visit `http://localhost/mini-bwapp`
5. Change security level at the top!
